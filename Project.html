WatchPick - 스마트 시계 추천 플랫폼
2024.01 ~ 2024.04 | 팀 프로젝트 | 팀장 · 프론트엔드/백엔드 개발 (5명)
프로젝트 개요
사용자의 취향을 분석해 최적의 스마트워치를 추천하는 플랫폼.
팀 프로젝트, 참여인원 5명(팀장, FE/BE 개발)
프로젝트 소개
목적/목표: 스마트워치 정보 과잉 문제 해결, 개인별 맞춤 추천 제공
문제 정의: 정보 탐색의 번거로움 → 자동화된 필터링 및 추천
주요 기능: 설문-추천, 상세 비교, 커뮤니티, 실시간 피드백
기술 스택: React, Node.js, AWS, Styled-components, MongoDB 등
상세 내용
진행 단계: 요구분석 → 설계 → 개발 → 배포
담당/기여: 메인 페이지, 추천 알고리즘, 배포 자동화
{ 추천로직 } User → 설문 → 추천엔진 → DB조회 → 결과 노출
기술적 도전: 사용자별 추천 DB 최적화, 비동기 처리 문제
해결: MongoDB 인덱싱, useEffect 최적화
결과 및 성과
성과: 실제 200명 유저 설문 및 추천, 배포
어려웠던 점: 데이터 불일치/최적화
극복: 테스트 자동화, 코드 리뷰
배포/깃허브: GitHub 바로가기
기타
차후 개선: 더 세밀한 필터링, 워치버전 추가
자료: 프로젝트 발표자료
기여도(본인): 메인 아키텍처 설계, 알고리즘 구현, 배포 총괄
